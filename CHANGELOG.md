## How to Update

1. Update version numbers
2. Add changes under appropriate section
3. Add [Unreleased] section at top
4. Create git tag with version number

Example tag:
```bash
git tag -a v1.1.0 -m "Release version 1.1.0"
git push origin v1.1.0
```
